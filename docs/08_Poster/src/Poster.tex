\section*{\color{PSUgreen}The Problem}
Ichor Systems generates large volumes of thick rigid plastic
(PVC, PP, LDPE) during semiconductor equipment manufacturing.
Off-the-shelf shredders are either too small for the material
(0.25--0.5\,in.~sheets, 4--12\,in.~wide) or priced far above the
sponsor's per-line budget. The EE team designed and built the
electrical control and power system for a new industrial-grade
plastic shredder. The mechanical structure and cutting blades are
delivered by a separate ME team.

\section*{\color{PSUgreen}Requirements}
\textbf{Must:}
\begin{itemize}\setlength{\itemsep}{2pt}
  \item Shred plastic to $\sim$1\,in.~pieces.
  \item Drive a 3--5\,HP motor at 70--90\,RPM through a controllable VFD.
  \item Follow NFPA 70 (NEC); include E-stop, overload, and one safety interlock.
  \item Provide an HMI for operator control and status.
  \item Stay within the \$1{,}000 EE-team budget.
\end{itemize}
\textbf{Should:} status / fault lights, NEMA enclosure, labeled controls. \\
\textbf{May:} read-only wireless monitoring, data logging, capacitive-touch safety.

\section*{\color{PSUgreen}System Architecture}
\begin{center}
\includegraphics[height=4.5in]{./Pictures/Photo_Front_Panel.jpg}\hspace{6pt}%
\includegraphics[height=4.5in]{./Pictures/Photo_Inside_Panel.jpg}\\[2pt]
{\small\itshape As-built operator control panel (left) and internal wiring (right): EA1-T4CL HMI, deadman / forward / reverse / E-stop pushbuttons, DirectLOGIC 205 rack, Huanyang VFD, Mean Well 24\,VDC supply, SD-N35 contactor, and the fused terminal blocks.}

The control system is built end-to-end on AutomationDirect industrial
hardware -- no hobby-grade microcontrollers are in the safety or
control path. A PowerTec 71008 disconnect feeds a Mean Well NDR-480-24
24\,VDC supply, a Mitsubishi SD-N35 contactor (E-stop hardwired in
series with its coil), and a Huanyang HY02D211B-T VFD. The VFD drives
a 3\,HP bench motor (GE 5KE182BC205B) over U/V/W via blue, white, and
red 4\,mm banana sockets.

\section*{\color{PSUgreen}PLC Control}
\begin{center}
\includegraphics[width=0.82\columnwidth]{./Pictures/PLC.png}\\[2pt]
{\small\itshape DirectLOGIC 205 rack with H2-DM1E CPU and the five I/O modules.}
\end{center}

The PLC is the system controller. A DirectLOGIC 205 D2-09B-1 rack
holds the H2-DM1E CPU and five I/O modules: \texttt{D2-08ND3}
(8\,DC inputs for pushbuttons, E-stop monitor, VFD fault, capacitive
touch), \texttt{F2-08AD-1} (analog current input for VFD VO motor
current), \texttt{F2-04RTD} (motor temperature, future),
\texttt{D2-08TR} (8 relay outputs for VFD RUN\_FWD / RUN\_REV /
RESET and panel LEDs), and \texttt{F2-08DA-2} (0--10\,V analog
output to the VFD VI speed reference). The 31-rung Do-more program
implements a deterministic state machine plus a 3-strike jam recovery
sequence and a hard-lockout path.

\section*{\color{PSUgreen}Safety \& Compliance}
\begin{itemize}\setlength{\itemsep}{2pt}
  \item \textbf{E-stop:} Category 0 stop. Cutler-Hammer E22B1 NC mushroom
        in series with the SD-N35 contactor coil. Drops the contactor
        independently of the PLC.
  \item \textbf{Two-hand / deadman:} both pushbuttons must be held to
        enable the motor (PLC X2 AND X3, with HMI soft mirrors C23 / C24).
  \item \textbf{Overcurrent / jam:} VFD over-torque trip (\texttt{PD124 =
        150\%}, \texttt{PD125 = 3\,s}) drives PLC X5 through the
        fault relay; ladder runs the 3-strike unjam routine.
  \item \textbf{Enclosure:} Vorksmen SEB001 steel, manufacturer-rated
        \textbf{NEMA Type 12} (dust-tight, drip-tight).
  \item \textbf{Compliance:} NFPA 70 (NEC), NFPA 79, NEMA 250 / ICS / MG~1,
        ISO 12100, ISO 13849-1 (preliminary Cat~B--1 / PL b--c), ISO 13850,
        ISO 13851, OSHA 29 CFR 1910.147 (LOTO).
\end{itemize}

\section*{\color{PSUgreen}Results}
\begin{itemize}\setlength{\itemsep}{2pt}
  \item Panel built, wired, and bench-validated. All 31 ladder rungs
        program and scan correctly.
  \item All four HMI screens operate; tag database matches the PLC.
  \item Speed reference path verified: HMI \texttt{V2000}
        $\rightarrow$ \texttt{SCALE} $\rightarrow$ \texttt{F2-08DA-2 CH1}
        $\rightarrow$ VFD \texttt{VI}.
  \item Jam-recovery verified: overcurrent on \texttt{X5} latches
        \texttt{C10} $\rightarrow$ 2\,s reverse (\texttt{T1})
        $\rightarrow$ coast-to-stop buffer (\texttt{T3}).
  \item Three-revision point list reviewed by Fearghus Tyler at each stage.
\end{itemize}
