# Prior Capstone Work (2024)

Reference materials from the **previous (2023–2024) PSU capstone
team** who built the first version of the Ichor Systems plastic
shredder. Imported from Google Drive at end of term so the sponsor
has a single repo with the complete project lineage.

| File | What it is |
|---|---|
| `Capstone_Report_2024.pdf` | The prior team's full capstone final report for the original 2 HP plastic shredder build. |
| `Expo_Poster_2024.pdf` | Their end-of-year expo poster ("Group F — Plastic Shredder"). |

These were the starting point for our Universal Plastic Shredder
**V2.0** work: same sponsor, same problem domain, scaled to
industrial-grade plastics (PVC / PP / LDPE 0.25–0.5 in.~sheets)
with a stronger drive train, a real PLC + HMI control system,
and proper NFPA 79 / NEMA-compliant safety architecture.

These files are **read-only reference**. Nothing in this folder is
maintained by our team.
