# Google Drive Source Archive

Snapshot of the `.docx` working drafts that were authored on Google
Drive earlier in the term. They are kept here for completeness only.

**These files are NOT the canonical version of any deliverable.**
At some point during Winter term the team moved every active
deliverable into the GitHub repo and stopped editing on Drive.
The Drive copies stopped receiving updates from that point on, so
the version in this folder is a snapshot of the last Drive state,
not the final state of the document.
