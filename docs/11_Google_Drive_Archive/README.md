# Google Drive Archive

Everything in this folder is a snapshot of the team's Google Drive
contents at end of term, imported into the repository so the sponsor
has one place that holds the full project history.

The team moved active editing into the GitHub repo partway through
the project, so the snapshots here are **historical reference only**
— they are not the canonical version of any deliverable that has a
counterpart in the rest of `docs/`.

## Layout

| Folder | What's inside |
|---|---|
| [`Meeting_Notes/`](Meeting_Notes/) | Working `.docx` notes from team, sponsor, and advisor meetings across Winter / Spring 2026 |
| [`Presentations/`](Presentations/) | Four PowerPoint decks (early review decks, the TED-talk 2 deck, the final 5-minute demo deck) |
| [`Prior_Work_2024/`](Prior_Work_2024/) | The previous (2023–2024) PSU capstone team's final report and expo poster (the starting point for V2.0) |
| [`Sources/`](Sources/) | `.docx` working drafts of the project proposal, team contract, test plan, and weekly progress reports |
| `Sponsor_Brief.pdf` | Ichor Systems' original scope brief for the V2.0 redesign |
